📋 Setup Instructions

  1. Download and install Node.js from nodejs.org
  2. Copy .env.example to .env and add your Discord bot token
  3. Open PowerShell in the project folder
  4. Run: npm install
  5. Run: npm run register-commands (one time)
  6. Run: npm run dev
  7. Visit http://localhost:8080 to see this dashboard live
